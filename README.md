# HTML-CSS-Basics-for-QA

**Описание задания:**
В рамках задания изучены и применены базовые элементы HTML и CSS. 

**Цель:**  
Освоить базовую структуру веб-страниц и стилизацию.

**Что было сделано:** 
- структура HTML-документа  
- работа с заголовками, списками, ссылками и изображениями  
- подключение и применение CSS-стилей  
- изменение заголовка вкладки браузера  
- базовое оформление страницы
